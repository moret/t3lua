DALua - Documentação

 

dalua.mutex

Módulo de exclusão mútua distribuída.

dalua.mutex.create(id, procs)

Descrição

Cria um objeto de exclusão mútua (Mutex) com o identificador especificado nos processos listados. Caso já exista, um evento de erro é gerado.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do Mutex string N/A
procs Tabela de processos table N/A

dalua.mutex.enter(id, func, ...)

Descrição

Requisita acesso à região crítica do Mutex especificado. Quando o acesso for obtido, func será chamada com os argumentos passados. Caso o identificador não exista, um evento de erro é gerado.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do Mutex string N/A
func Função a ser chamada function N/A
... Argumentos a serem passados para a função variável nil

dalua.mutex.leave(id)

Descrição

Libera o acesso à região crítica do Mutex especificado.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do Mutex string N/A

dalua.mutex.destroy(id)

Descrição

Destrói o objeto de exclusão mútua especificado, caso exista.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do Mutex string N/A

dalua.mutex.add(id, proc)

Descrição

Adiciona um processo ao grupo de processos do Mutex especificado. Esta função deve ser chamada por um processo que já participa do Mutex, quando não estiver na região crítica.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do Mutex string N/A
proc Identificador do processo a ser adicionado string N/A

dalua.mutex.remove(id, proc)

Descrição

Remove um processo do grupo de processos do Mutex especificado. Esta função deve ser chamada por um processo que já participa do Mutex (podendo inclusive ser o próprio removido), quando não estiver na região crítica.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do Mutex string N/A
proc Identificador do processo a ser removido string N/A
