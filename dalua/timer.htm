DALua - Documentação

 

dalua.timer

Módulo do sistema de temporizadores. Permite executar tarefas periodicamente no processo local ou em processos remotos.

dalua.timer.add(id, period, count, func, ...)

Descrição

Cria um temporizador no processo especificado. A função func será chamada count vezes a cada duration segundos. Se count for igual a zero, o temporizador só terminará após ser removido.

Retorna o identificador do temporizador.

Parâmetros

Nome Valor Tipo Padrão
id Identificador do processo string N/A
period Intervalo em segundos entre as chamadas número N/A
count Número de vezes, zero para infinito número N/A
func String com o nome da função a ser chamada string N/A
... Argumentos a serem passados para a função variável nil

dalua.timer.remove(timer)

Descrição

Remove o temporizador especificado.

Parâmetros

Nome Valor Tipo Padrão
timer Identificador do temporizador número N/A
