DALua - Documentação

 

dalua.app

Módulo de suporte a aplicações. Cada aplicação é um grupo lógico de processos.

 

dalua.app.init()

Descrição

Inicializa o sistema de aplicações DALua no daemon corrente. Deve ser chamado por cada processo antes de se utilizar as demais funções do módulo dalua.app.

dalua.app.create(app)

Descrição

Cria uma aplicação DALua com o nome especificado na rede conhecida de daemons. Caso já exista, um evento de erro é gerado.

Parâmetros

Nome Valor Tipo Padrão
app Nome da aplicação string N/A

dalua.app.join(app)

Descrição

Entra na aplicação especificada. Caso não exista, um evento de erro é gerado.

Parâmetros

Nome Valor Tipo Padrão
app Nome da aplicação string N/A

dalua.app.processes(app)

Descrição

Retorna uma tabela contendo os identificadores de todos os processos de uma aplicação.

Parâmetros

Nome Valor Tipo Padrão
app Nome da aplicação string N/A

dalua.app.applications()

Descrição

Retorna uma tabela contendo os nomes das aplicações de que o processo atual participa.

dalua.app.leave(app)

Descrição

Sai da aplicação especificada.

Parâmetros

Nome Valor Tipo Padrão
app Nome da aplicação string N/A

dalua.app.destroy(app)

Descrição

Desfaz a aplicação especificada.

Parâmetros

Nome Valor Tipo Padrão
app Nome da aplicação string N/A

dalua.app.link(addr, port)

Descrição

Possibilita o uso de aplicações distribuídas em diferentes daemons. Esta função deve ser chamada após o 'dalua.link' completar com sucesso, utilizando o mesmo endereço e porta.

O efeito é que as aplicações que existem na rede em que se conectou tornam-se acessíveis aos processos do daemon local e vice-versa. Caso o daemon local tenha aplicações com mesmo nome, elas serão sobrescritas.

Diferentemente do 'dalua.link', esta chamada atinge apenas o daemon do processo que chamou a função, e não a rede completa.

Parâmetros

Nome Valor Tipo Padrão
addr Endereço IP do daemon a se conectar string N/A
port Porta do daemon a se conectar número N/A
